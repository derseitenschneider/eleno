Key Differences from Frontend

1. Color Format

Backend uses hex colors (#4794ae) while frontend uses HSL values (hsl(195, 42%, 48%))

2. No Dark Mode

The homework pages don't include dark mode support

3. Traditional CSS

Uses traditional CSS classes instead of Tailwind utilities

4. Simplified Components

Only basic components needed for homework display (no complex UI elements)

Future Migration Path

Recommendations for aligning backend styles with the main design system.

Suggested Improvements

  1. Convert colors to HSL - Match the frontend color format for consistency
  2. Extract shared variables - Create a shared design tokens file both can import
  3. Add CSS custom properties - Use CSS variables for easier theming
  4. Consider Tailwind CSS - Use Tailwind's CDN for the PHP views
  5. Implement dark mode - Add theme support for consistency

Note: Any changes should maintain backward compatibility with existing homework share links.
