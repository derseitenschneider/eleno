Usage Guidelines

Consistency

Use the same interaction patterns throughout the application. If buttons scale on click in one place, they should behave the same way everywhere.

Performance

Use CSS transitions and transforms for better performance. Avoid animating properties like width, height, or position when possible.

Accessibility

Respect prefers-reduced-motion settings. Provide clear focus indicators for keyboard users. Don't rely solely on color to convey state changes.

Feedback Timing

Provide immediate feedback for user actions. Show loading states for operations that take more than 300ms. Use skeleton screens for content that's loading.
