Design Principles

Clarity

Every element should have a clear purpose and be easily understood by teachers using the platform.

Consistency

Maintain visual and behavioral consistency across all features to reduce cognitive load.

Accessibility

Design for all users with proper contrast ratios, keyboard navigation, and screen reader support.

Mobile-First

Optimize for mobile devices while ensuring a great desktop experience.
