// Default button
<Button>Click me</Button>

// With variant and size
<Button variant="outline" size="lg">
  Large Outline Button
</Button>

// Destructive action
<Button variant="destructive">
  Delete Item
</Button>

// As a link
<Button variant="link" asChild>
  <a href="/dashboard">Go to Dashboard</a>
</Button>

// With loading state
<Button disabled>
  <Loader2 className="mr-2 h-4 w-4 animate-spin" />
  Please wait
</Button>

Usage Guidelines

Primary Actions

Use the default variant for primary actions like "Save", "Submit", or "Create". Limit to one primary button per section to maintain clear hierarchy.

Destructive Actions

Always use the destructive variant for delete operations. Consider adding confirmation dialogs for irreversible actions.

Button Text

Use clear, action-oriented text. Start with a verb when possible (e.g., "Create Lesson" instead of "New"). Keep text concise, ideally 2-3 words.

Accessibility

All buttons include focus states for keyboard navigation. Ensure sufficient color contrast and provide descriptive labels for icon-only buttons using aria-label.
