Body Text

Standard text styles for paragraphs, labels, and UI elements.

Body · base (16px)

This is standard body text used throughout the application. It provides excellent readability for lesson content, descriptions, and general interface text. The base size ensures comfortable reading on both desktop and mobile devices.

Small · sm (14px)

Small text is used for secondary information, timestamps, and metadata. It maintains readability while creating visual hierarchy.

Extra Small · xs (12px)

Extra small text for captions, labels, and tertiary information.

Muted · text-muted-foreground

Muted text color is used for placeholder text, helper text, and less important information that shouldn't compete with primary content.

Usage Guidelines

Line Height

Body text uses a relaxed line height for better readability. Headings use tighter line heights to create visual cohesion in larger text.

Responsive Typography

Font sizes remain consistent across breakpoints to maintain readability. Layout adjustments handle responsive design rather than scaling typography.

Content Hierarchy

Use the heading hierarchy consistently. Don't skip heading levels, and use semantic HTML elements (h1-h6) for proper document structure and accessibility.
