Usage Guidelines

Accessibility

All color combinations meet WCAG 2.1 AA standards for contrast ratios. Primary colors on their respective foregrounds achieve a minimum 4.5:1 ratio.

CSS Variables

Colors are defined as CSS custom properties and can be used with Tailwind's arbitrary value syntax:

bg-[hsl(var(--primary))]

Semantic Usage

Use semantic color tokens (primary, secondary, destructive) rather than specific color values to ensure consistency and theme support.
