Usage Guidelines

Consistency

Stick to the spacing scale. Avoid arbitrary values like p-[17px]. If you need a specific size, round to the nearest scale value.

Vertical Rhythm

Use consistent vertical spacing with utilities like space-y-4 for stacked elements rather than individual margin classes.

Component Spacing

Components should handle their internal spacing. Parent containers control spacing between components.

Mobile Considerations

Reduce spacing on mobile to maximize content area. Use responsive modifiers like p-4 sm:p-6 to adjust padding at breakpoints.
